## Streamlit Keepalive

This repo includes `.github/workflows/streamlit-keepalive.yml`, which pings the Streamlit app every 6 hours to reduce Community Cloud hibernation.

Optional: set a repository variable named `STREAMLIT_APP_URL` if your app URL changes.  
Default fallback is `https://zoswiai.streamlit.app`.
